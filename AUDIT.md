# Отчет об аудите проекта "Глитчи Реальности"

## Общая структура

- Статический сайт: множество `.html` страниц и ресурсы в `assets` и `content`.
- Основные стили и скрипты находятся в `styles.css` и `scripts.js`.

## Тесты

В каталоге `tests/` есть `progress.test.js`, запускаемый через Jest.

## Проверка качества кода

- `htmlhint` не выявил ошибок в HTML-файлах.
- `eslint` и `stylelint` не настроены.

## Возможные улучшения

1. Вынести общие CSS и JS в отдельные файлы.
2. Добавить сборщик для минимизации ресурсов.
3. Настроить ESLint и расширить тесты.
4. Проверить доступность: `alt` для изображений и навигация с клавиатуры.
5. Хранить критичные зависимости локально.
