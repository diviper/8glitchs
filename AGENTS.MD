# Project Agent Guide — Glitch Registry

## Goals
- Убрать фикс «8». Проект — открытый реестр **Glitch Registry**.
- Структура:
  - `/scenes/*.html` — интерактивы.
  - `/content/glitches/*.md` — карточки глитчей (Markdown по шаблону).
  - `/legacy/` — архив старых страниц/ассетов.
  - `/assets/js/{router.js, progress.js, lexicon.js}` — роутер, прогресс, словарь.
- Обновить `README.md` под реестр без нумерации, со списком slug’ов.

## Naming
- Сцены: `scenes/glitch-<slug>.html` (латиница, дефисы).
- Карточки: `content/glitches/<slug>.md`.
- Всё `reality_bug_*` перенести в `/legacy/` (ссылки обновить в `README.md`).

## Markdown rules
Каждая карточка содержит секции:
1. `### TL;DR`
2. `### Научная опора`
3. `### Парадокс`
4. `### Дневник Дивайпера`
5. `### Юнг`
6. `### Сенека`
7. `### Рик`
8. `### Сцена/механика`
9. `### Ссылки`

Правила: заголовки ≤ H3, короткие абзацы, ссылки с короткими пояснениями.
