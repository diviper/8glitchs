# Glitch Registry

Открытый реестр глитчей реальности: интерактивные сцены и карточки.

## Структура

- `scenes/` — интерактивы
- `content/glitches/` — карточки глитчей
- `legacy/` — архив старых страниц
- `assets/` — статические ресурсы
